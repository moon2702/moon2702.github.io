Linux 命令 用户相关

发表于2024-08-08|学习经历

文件所有者和用户组

chown 和 chgrp

常用选项

选项 作用
-R
–recursive		 递归改变目录子项的所有者和组
–reference=[指定文件] 将文件或者目录的所有者和组设置为和指定文件相同
-h
–no-derefernece		 操作链接文件本身，而不是它指向的文件

举例

用户管理

命令 用法 作用
whoami whoami 查看当前用户
who who 查看当前登录系统的用户信息
w w 查看当前登录系统的用户及其活动
last last 显示用户登录历史
su su [用户名] 切换用户
passwd passwd [用户名] 修改用户密码
chfn chfn [用户名] 修改用户的详细信息
chsh chsh [用户名] 修改用户登录Shell
adduser 添加新的用户
usermod 修改用户配置
userdel 删除某用户
chage 修改和查看用户密码配置
id 显示用户和组的信息

adduser

常用选项

选项 作用
–home [目录名] 指定用户的主目录，默认 /home/username
–shell [SHELL] 指定用户的登录shell，默认 /bin/bash
–no-create-home 不创建用户的主目录
–uid [用户ID] 指定用户的用户ID
–gid [用户组] 指定用户的初始组ID(组名),默认创建同名用户组
–disabled-password 创建一个没有密码的账户
–gecos [GECOS] 为用户指定GECOS字段的值

usermod

常用选项

选项 作用
-c
–comment		 更改用户GECOS字段
-d
–home		 更改用户主目录
-e
–expiredate		 设置账户过期时间
-f
–inactive		 设置密码过期后多少天账户被禁用
-g
–gid		 更改用户初始登录组
-G
–groups		 更改用户所属附加组
-l
–login		 更改用户名
-L
–lock		 锁定用户密码，使其无法登录
-m
–move-home		 移动用户主目录到新位置
-p
–password		 设置用户的密码
-s
–shell		 更改用户登录shell
-u
–uid		 更改用户ID
-U
–unlock		 解锁用户密码，使其可以登录

userdel

常用选项

选项 作用
-r
–remove		 删除用户主目录和内容
-f
–force		 强制删除用户，即使用户已经登录或运行进程
-Z
–selinux-user		 指定删除用户时的SELinux用户标签

问题

如果系统使用SELinux，可以使用-Z指定在删除用户时应用的SELinux用户标签，这是为了确保在删除用户时也正确更新SELinux上下文。

chage

用于修改和查看用户密码的有效期信息。可以帮助管理员控制密码过期时间和用户需要修改密码的频率。

常用选项

选项 作用
-l 显示用户密码的有效期信息
-d 设置用户上次更改密码的日期
-E 设置用户账户过期日期
-I 设置密码过期后多少天账户被禁用
-m 设置密码最短使用期限(多少天内不能更改)
-M 设置密码最长使用期限(多少天后必须更改)
-W 设置密码过期前多少天警告用户
-h 显示帮助信息

id

id命令常用于显示用户和组的信息。可以显示当前用户或者指定用户的UID，GID和用户所属的附加组。

常用选项

选项 作用
-u 显示用户UID
-g 显示用户基本组GID
-G 显示用户所属附加组GID
-n 显示用户和组的名字，而不是ID
-r 显示真实用户ID，而不是有效用户ID
-Z 显示SELinux上下文

用户组管理

命令 用法 作用
newgrp newgrp [组名] 切换有效用户组
groups groups [用户名] 查看当前用户所属组
groupdel group [组名] 删除一个组
groupadd 添加一个新的组
gpasswd 管理组密码文件
groupmod 修改组的配置，组名、ID等

groupadd

常用选项

选项 作用
-g 指定新组的组ID
-r 创建一个系统组，GID小于1000
-o 允许使用重复的GID

gpasswd

用于管理组密码文件，并进行组管理员相关操作。

常用选项

选项 作用
-a
–add		 添加用户到组
-d
–delete		 从组中删除用户
-r
–remove-password		 删除组密码
-R
–restrict		 禁止组成员自行切换到组
-A
–administrators		 设置组管理员
-M
–members		 定义组成员

问题

  1. 什么是禁止组成员自行切换到组？
    一般来说用户可以通过 newgrp [组名] 来切换到另一个组，从而影响用户对文件和目录的访问权限，当通过 gpasswd -R [组名] 命令之后，非组成员用户无法通过 newgrp [组名] 来切换到该组。

groupmod

用于修改组的配置，例如组名、组ID

常用选项

选项 作用
-g 修改组的GID
-n 修改组名
-o 允许使用重复的GID
